# Diff Studio Flutter

Windows desktop rebuild of Diff Studio using Flutter.

## Run on Windows

Install Flutter for Windows first, then run:

```powershell
cd diff-studio-flutter
.\windows_setup.ps1
.\run_windows.ps1
```

On a fresh Windows VM, you can try installing the build environment automatically first:

```powershell
cd diff-studio-flutter
powershell -ExecutionPolicy Bypass -File .\install_windows_build_env.ps1
```

After it finishes, close and reopen PowerShell, then build the app.

Build a release package:

```powershell
.\build_windows.ps1
```

The executable folder and zip package will be generated under:

```text
dist\DiffStudio-Windows\
dist\DiffStudio-Windows.zip
```

The app executable is:

```text
dist\DiffStudio-Windows\diff_studio_flutter.exe
```

If PowerShell blocks scripts on a fresh Windows VM, run this once in the project folder:

```powershell
powershell -ExecutionPolicy Bypass -File .\build_windows.ps1
```

## Build with GitHub Actions

If you do not have a Windows build machine, upload this project to a private GitHub repository and run the `Build Windows` workflow from the Actions tab.

The workflow will produce an artifact named:

```text
DiffStudio-Windows
```

Download and unzip it to get:

```text
DiffStudio-Windows.zip
diff_studio_flutter.exe
```

## Current Windows Features

- Left/right file picking.
- Left/right drag-and-drop targets.
- Text/code/JSON diff.
- PDF text extraction.
- Word `.docx` text extraction.
- Excel `.xlsx`, CSV, and TSV table comparison.
- Independent left/right horizontal scrolling for table comparison with a fixed center divider.
- Image side-by-side preview.
- Binary file size summary.
- Difference sidebar with line jumps.
- Ignore whitespace, case, blank lines, and only-differences view.
- Translucent glass-style Windows desktop UI.

## Notes

This repo intentionally keeps the Flutter source portable. The `windows` runner folder is generated by `flutter create --platforms=windows .` on the Windows machine that will build the app.
